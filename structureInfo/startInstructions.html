Instructions

Image navigation should be clear from the display to the left.
There are also some keyboard commands:
KeyFunction
rReset View (i.e. pan, zoom, level, window)
SpaceHide/Show structures
w/a/s/dPan in given direction
W/A/S/DPan in given direction (faster)
Up/DownSlice up/down
i/oZoom in/out
eToggle edges only
bToggle bone window
hToggle this help (including image)

Structures can be highlighted from the drop-down menu
at the top (click in the check box).

Scrolling through slices can also be donw with the mouse
scroll wheel. Holding control while scrolling will zoom.
